# AWS Secure Serverless Application

A secure serverless notes API built with AWS CDK, API Gateway, Lambda, DynamoDB, IAM, and CloudWatch.

## Project Goal

This project demonstrates AWS cloud engineering skills through a small but realistic serverless application.

## Version 1 Scope

- Create notes
- View notes
- Delete notes
- Store data in DynamoDB
- Deploy infrastructure using AWS CDK

## AWS Services Used

- API Gateway
- AWS Lambda
- Amazon DynamoDB
- IAM
- CloudWatch
- AWS CDK

## Architecture

User → API Gateway → Lambda → DynamoDB
